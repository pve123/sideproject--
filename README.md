# Springboot side project ( JWT + Redis + Gradle + Swagger + SMTP + DDD + Entity Migration + JPA + QueryDSL + MySQL + Docker + Jasypt + WebSocket)


### summary : 회원 + 게시판 + 댓글 + 이메일 인증 + 랜덤채팅

- [x] 회원가입 
- [x] 로그인(JWT)

- [x] 회원탈퇴
- [x] 비밀번호 변경
- [x] 임시 비밀번호 발급(SMTP, Redis)
- [x] 게시글 작성
- [x] 게시글 수정
- [x] 게시글 삭제
- [x] 이미지 업로드
- [x] 이미지 삭제
- [x] 게시글 조회
- [x] 게시글 조회 수
- [x] 게시글 목록 (Paging)
- [x] 회원 조회
